# Steuern

## 1. Ermittlung der Gewerbesteuerschuld

### 1.1 Grundlagen & Steuerpflicht

Die **Gewerbesteuer** steht den Gemeinden zu und dient der Bestreitung ihrer Aufgaben. Steuerpflichtig ist – unabhängig von der Rechtsform – **jeder Gewerbetreibende**, also Einzelunternehmer, juristische Personen und Personengesellschaften.

---

> [!IMPORTANT]
> **Wichtige Aufgabenteilung:**
>
> - Das **Finanzamt** setzt die Besteuerungsgrundlage (Messbetrag) im **Gewerbesteuermessbescheid** fest.
> - Die **Gemeinde** legt den **Hebesatz** fest und erlässt den eigentlichen **Gewerbesteuerbescheid**.

---

### 1.2 Berechnungsschema (Schritt für Schritt)

| Schritt                                                                                                              | Berechnung                    |
| -------------------------------------------------------------------------------------------------------------------- | ----------------------------- |
| Gewinn (Einkünfte) aus dem Gewerbebetrieb                                                                            | Ausgangsgröße                 |
| + Hinzurechnungen                                                                                                    | z. B. Zinsen, Mieten, Pachten |
| − Kürzungen                                                                                                          | z. B. Grundstückserträge      |
| = **Maßgebender Gewerbeertrag**                                                                                      |                               |
| − evtl. Gewerbeverlust (Vorjahre)                                                                                    |                               |
| = **Gewerbeertrag** (abgerundet auf volle 100 EUR)                                                                   |                               |
| − Freibetrag **24.500 EUR** _(nur Einzelunternehmer &amp; Personengesellschaften, nicht bei Kapitalgesellschaften!)_ |                               |
| = **Gewerbeertrag nach Freibetrag**                                                                                  |                               |
| × Steuermesszahl **3,5 %**                                                                                           |                               |
| = **Steuermessbetrag**                                                                                               |                               |
| × **Hebesatz** der Gemeinde (mind. 200 %)                                                                            |                               |
| = **Gewerbesteuerschuld**                                                                                            |                               |

### 1.3 Wichtige Einzelheiten

- Der **Steuermessbetrag** wird vom Finanzamt ermittelt und sowohl dem Steuerpflichtigen als auch der Gemeinde mitgeteilt.
- Der **Hebesatz** ist vom Gemeinde-/Stadtrat festgesetzt und nach oben offen. Der **Mindesthebesatz beträgt 200 %**.
- Es werden **vierteljährliche Vorauszahlungen** auf die Gewerbesteuer festgesetzt.
- Die Gewerbesteuer ist **nicht als Betriebsausgabe abzugsfähig**, es ist jedoch eine **Gewerbesteuerrückstellung** zu bilden.
- Die **Gewerbesteuer wird auf die Einkommensteuer angerechnet**: Die tarifliche Einkommensteuer ermäßigt sich um das **4,0-Fache des Gewerbesteuer-Messbetrags**.

### 1.4 Rechenbeispiel

---

> [!NOTE]
> **Annahmen:**
> Einzelunternehmer / Gewinn = 100.000 EUR / Hebesatz = 400 %

---

$$\text{Gewerbeertrag} = 100.000{,}00\ \text{EUR}$$

$$\text{Freibetrag} = -24.500{,}00\ \text{EUR}$$

$$\text{Gewerbeertrag nach Freibetrag} = 75.500{,}00\ \text{EUR}$$

$$\text{Steuermessbetrag} = 75.500 \times 3{,}5\% = 2.642{,}50\ \text{EUR}$$

$$\text{Gewerbesteuerschuld} = 2.642{,}50 \times 400\% = 10.570{,}00\ \text{EUR}$$

### 1.5 Fehlerhafte Bescheide – Was tun?

Ist der **Steuermessbescheid fehlerhaft**, muss er durch:

- **Einspruch** bei der Finanzverwaltung (Finanzamt) **oder**
- **Klage** bei den Finanzgerichten

angefochten werden.

---

> [!IMPORTANT]
> **Prüfungshinweis:** Rechtsmittel sind beim **Finanzamt** (Steuermessbescheid) einzulegen – **nicht** bei der Gemeinde (die nur den Hebesatz festsetzt)!

---

## 2. Anfechtung des Einkommensteuerbescheids

### 2.1 Was ist der Einkommensteuerbescheid?

Das Finanzamt erlässt einen **Einkommensteuerbescheid**, der ausweist:

- Das veranlagte Einkommen
- Die der Tabelle entnommene Jahressteuer
- Bereits verrechnete Lohnsteuern und geleistete Vorauszahlungen

Der Bescheid kann **endgültig** oder **vorläufig** (unter Vorbehalt der Nachprüfung) ergehen.

### 2.2 Der Rechtsweg – 3 Stufen

```

Steuerbescheid erhalten
│
▼
┌───────────────────────────────────────────────────┐
│ STUFE 1: EINSPRUCH beim zuständigen Finanzamt     │
│ • Schriftlich, elektronisch oder zur Niederschrift│
│ • Kostenfrei                                      │
│ • Frist: 1 Monat ab Bekanntgabe des Bescheids     │
│  Das Finanzamt darf den GESAMTEN Bescheid prüfen  │
└───────────────────────────────────────────────────┘
│ Bei negativer Entscheidung
▼
┌───────────────────────────────────────────────────┐
│ STUFE 2: KLAGE beim Finanzgericht                 │
│ • Bei negativem Ausgang: kostenpflichtig          │
│ • Frist: 1 Monat ab Bekanntgabe des Urteils       │
└───────────────────────────────────────────────────┘
│ Bei negativer Entscheidung
▼
┌───────────────────────────────────────────────────┐
│ STUFE 3: REVISION beim Bundesfinanzhof (BFH)      │
│ • Nur bei Zulassung möglich                       │
│ • Bei negativem Ausgang: kostenpflichtig          │
│ • Frist: 1 Monat (Einlegung) / 2 Monate (Begr.)   │
└───────────────────────────────────────────────────┘

```

### 2.3 Wichtige Fristen & Regelungen

| Thema                   | Regelung                                                        |
| ----------------------- | --------------------------------------------------------------- |
| **Einspruchsfrist**     | 1 Monat ab Bekanntgabe des Bescheids                            |
| **Zustellung per Post** | Gilt am **4. Tag** nach Aufgabe zur Post als zugestellt         |
| **Steuerzahlung**       | Einlegung des Rechtsbehelfs schiebt Zahlung **NICHT** auf!      |
| **Säumniszuschlag**     | Kann bei Nichtbezahlung festgesetzt werden                      |
| **Empfehlung**          | Gleichzeitig **Antrag auf Aussetzung der Vollziehung** stellen! |

---

### 2.4 Verjährungsfristen (Abgabenordnung)

**Festsetzungsverjährung** (Finanzamt kann Steuer nicht mehr festsetzen):

| Steuerart / Situation          | Frist        |
| ------------------------------ | ------------ |
| Zölle und Verbrauchsteuern     | 1 Jahr       |
| Übrige Steuern (Regelfall)     | **4 Jahre**  |
| Leichtfertig verkürzte Steuern | 5 Jahre      |
| Hinterzogene Steuern           | **10 Jahre** |

**Zahlungsverjährung** (Finanzamt kann festgesetzte Steuer nicht mehr einfordern):

| Situation           | Frist        |
| ------------------- | ------------ |
| Regelfall           | **5 Jahre**  |
| Steuerhinterziehung | **10 Jahre** |

---

> [!NOTE]
> Die Verjährungsfrist beginnt mit Ablauf des Kalenderjahres, in dem der Anspruch erstmals fällig wurde.

---

### 2.5 Prüfungsrelevante Merksätze

---

> [!IMPORTANT]
>
> - **Richtiges Rechtsmittel:** Gegen einen Einkommensteuerbescheid ist der **Einspruch beim zuständigen Finanzamt** einzulegen – **nicht** beim Zivilgericht, Verwaltungsgericht oder direkt beim Finanzgericht!
> - **Keine Aufschiebung:** Die Einlegung eines Rechtsbehelfs schiebt die **Steuerzahlung nicht auf** – deshalb immer zusätzlich **Aussetzung der Vollziehung** beantragen!
> - **Gesamtprüfung:** Mit dem Einspruch berechtigt man das Finanzamt zur Prüfung des **gesamten** Steuerbescheids – auch zuungunsten des Steuerpflichtigen!
> - **Einspruch ist kostenlos** – Klage und Revision sind bei negativem Ausgang **kostenpflichtig**!

## 3. Schnellübersicht: Gewerbesteuer vs. Einkommensteuer

| Merkmal               | Gewerbesteuer                                | Einkommensteuer                                  |
| --------------------- | -------------------------------------------- | ------------------------------------------------ |
| Empfänger             | Gemeinde                                     | Bund/Länder                                      |
| Bescheid durch        | Gemeinde (Hebesatz) / Finanzamt (Messbetrag) | Finanzamt                                        |
| Besteuerungsgrundlage | Gewerbeertrag                                | Zu versteuerndes Einkommen                       |
| Freibetrag            | 24.500 EUR (Einzelunternehmer/PersGes)       | Grundfreibetrag                                  |
| Steuersatz            | Messzahl 3,5 % × Hebesatz                    | 14 % – 45 % (progressiv)                         |
| Rechtsmittel          | Einspruch beim Finanzamt                     | Einspruch beim Finanzamt                         |
| Vorauszahlungen       | Vierteljährlich                              | Vierteljährlich (10.03., 10.06., 10.09., 10.12.) |
